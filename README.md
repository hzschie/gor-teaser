# Reichenau Memorial Website

A memorial website dedicated to the victims of the Arbeitserziehungslager Reichenau (Labor Education Camp Reichenau), which operated from 1941 to 1945 in Innsbruck, Austria. This site serves as both a digital memorial and educational resource about this Nazi camp where approximately 8,500 people were imprisoned and 114 lives were lost.
